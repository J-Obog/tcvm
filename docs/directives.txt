REFERENCE
---------------------------------------------------
<alpha> := (A ... Z) | (a ... z) 
<digit> := (0 ... 9)
<number> := <digit> {<digit>} 
<identifier> := (_ | <alpha>) {(_ | <alpha> | <digit>)}
<typespec> := byte | word | dword 
---------------------------------------------------


TEXT LABEL
---------------------------------------------------
label <identifier>
---------------------------------------------------


DATA DECLARATION
---------------------------------------------------
data [<identifier>] <typespec> <number>
resv [<identifier>] <number>
---------------------------------------------------
