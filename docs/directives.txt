REFERENCE
---------------------------------------------------
<alpha> := (A ... Z) | (a ... z) 
<digit> := (0 ... 9)
<number> := <digit> {<digit>} 
<identifier> := (_ | <alpha>) {(_ | <alpha> | <digit>)}
<datadef> := byte | word | dword | space
---------------------------------------------------


TEXT LABEL
---------------------------------------------------
label <identifier>
---------------------------------------------------


DATA DECLARATION
---------------------------------------------------
data [<identifier>] <datadef> <number>
---------------------------------------------------
